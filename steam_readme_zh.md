全模式全物品存档，内含包括 v1.4.5 新增物品在内的全部物品。
[b]请确保你的地图是最新的（v3.1.0），如果不是，请更新你的地图。[/b]

[h1]功能介绍[/h1]
[list]
[*]按物品 ID 排序；（使用编码传送，[u][url=https://terraria.wiki.gg/zh/wiki/%E7%89%A9%E5%93%81_ID]物品ID列表[/url][/u]）
[*]按物品类型分类； [b](新!)[/b]（v1.4.5新增物品已用红色墙标识）
[*]NPC全图鉴； [b](新!)[/b]（Book Worm成就解锁）
[*]天空、地表、洞穴、地狱刷怪场；
[*]全自动普通、宝石树场；
[*]全城镇NPC房间
[*]全制作站
[/list]
