全模式全物品存档，内含包括 v1.4.5 新增物品在内的全部物品

[b]功能介绍[/b]
[list]
[*]按物品 ID 排序；（使用编码传送）
[*]按物品类型分类 (新!)；（v1.4.5新增物品已用红色墙标识）
[*]NPC全图鉴 (新!)；（Book Worm成就解锁）
[*]天空、地表、洞穴、地狱刷怪场；
[*]全自动普通、宝石树场；
[*]全城镇NPC房间
[*]全制作站
[/list]
